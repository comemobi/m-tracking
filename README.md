# 기능 요약

## 1. 통계 탭
- DPS, 총 데미지, 경과 시간, 타격 수, 치명/강타/연속/추가율 등 기본 통계 제공
- 스킬별 통계: 데미지, 히트, 최대/최소 데미지, 비율 바 표시
- 스킬 상세: 속성별 데미지, 특수 공격/브레이크/실드 등 세부 항목
- 대상(타겟) 선택 필터: 전체/특정 대상 기준 스킬 통계 보기
- 한국식 수치 표기(만/억/조)

## 2. 확장 통계
- 속성별 데미지, 데미지 분류, 실드, 특수 공격, 브레이크, 방어/회복 등
- 섹션 접기/펼치기 지원

## 3. 버프 통계
- 버프별 유지시간, 유지비율, 스택 통계 표시

## 4. 추적 탭
- 스킬/버프/타겟 스키마 조회
- 등록/수정/삭제 (모달 기반)
- 미식별 항목 추적 및 빠른 등록
- 검색/필터 지원

## 5. 로그 탭
- 실시간 패킷 추적 로그 확인
- 스킬/버프/타겟 로그 필터
- 로그 지우기

## 6. 히스토리 탭
- 이전 세션 목록 보기
- 상세 보기 시 통계 탭과 동일한 전체 통계 제공

## 7. 세션 제어
- 시작/중지/초기화
- 타이머(초 단위) 설정 시 자동 종료
- 자동 저장 옵션 (설정 탭)
- 세션 진행 시간 표시

## 8. 오버레이 탭
- 오버레이 표시/숨김
- 투명도 조절
- 키 맵핑(F8~F11 기본)
  - 숨기기/보이기, 시작/중지, 리셋, 이동 모드
- 버프 선택(지속시간/쿨타임 있는 버프만)
- 선택된 버프만 오버레이 표시

## 9. 설정 탭
- 테마(다크/라이트)
- 서버 상태/재시작
- 데이터 폴더 열기
- 자동 저장
- 디버그 모드(미식별 스킬/버프 표시)

## 10. 서버 연동 구조(간략)
- 로컬 Go 파싱 서버가 패킷을 캡처하고 실시간 통계를 계산
- WebSocket으로 통계/스킬/버프/타겟 데이터를 Electron 앱에 전송
- HTTP API로 스키마(스킬/버프/타겟) 조회 및 수정
- 세션 단위 기록 및 히스토리 조회 지원

## 11. 준비물(Windows)
- 패킷 캡처를 위해 Npcap 설치 필요
- 앱 시작 시 Npcap 미설치/미실행이면 안내 화면 제공
- 안내 화면에서 다운로드 페이지로 이동 가능
